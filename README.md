## Overview

A clean, professional, dark-themed portfolio built with **Next.js 15 + TypeScript**.  
It highlights my skills, learning journey, projects, and ambitions in enterprise architecture, AI-assisted flows, and SaaS integrations.

**Color Scheme:** Blue · Cyan · White · Black

## Getting Started

```bash
# Install dependencies
npm install

# Copy env vars
cp .env.example .env.local
# Fill in your values in .env.local

# Run dev server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

---

## Deployment

Deployed via **Vercel**. Push to `main` triggers automatic deployment.

```bash
vercel --prod
```
